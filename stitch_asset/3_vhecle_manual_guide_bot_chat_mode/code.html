How do I change the smart key battery?

Sent • 10:24 AM

To change the smart key battery in your 2024 Tucson Hybrid, follow these steps:

  1. Gently pry the key case open using a flat-head screwdriver.
  2. Remove the old battery (CR2032).
  3. Insert the new battery with the "+" terminal facing up.
  4. Snap the cover back together.
